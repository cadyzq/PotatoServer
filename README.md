# PotatoServer

## Quick Start Guide

### Front-End

`cd ./view`  
`npm install`  
`npm start`

### Back-End

`cd ./model`  
`npm install`  
`npm start`

- Express from npm express
- react-router-dom for routing
- react-hot-loader on babel
- nodemon
